# Student Mental Health Check-In App

A simple and responsive web application that helps students track their daily mental well-being. The app allows users to log mood, stress, sleep, and focus, and provides AI-based insights with supportive recommendations.

## Features
- Daily mood and stress check-in
- Sleep and focus tracking
- AI-based mental state analysis
- Personalized wellness tips
- Weekly and monthly progress charts
- Light and Dark mode
- Fully responsive for all devices

## Purpose
This project is designed to help students become more aware of their mental health patterns and manage academic stress in a healthy way.

## Technologies Used
- No-code / Low-code platform (VibeCode)
- AI-based text and pattern analysis
- Responsive UI components

## Ethical Note
This application does not provide medical diagnosis. It is intended only for mental well-being support and awareness.

## How to Use
1. Open the application
2. Complete the daily check-in form
3. View insights and recommendations
4. Track progress on the dashboard
